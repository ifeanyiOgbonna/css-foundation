The basic Language of the Web: HTML

posted by laura Jones on monday,June 21st 2027

All modern websites and web applications are built using three fundamental technologies: HTMl;CSS and Javascript.These are the language of the web.

in this post,let's focus on HTML. we will learn what HTML is all about,and why you too should learn it.

what is HTML?

HTML stand for HyperText Markup Language. it's a markup language that web developers use to structure and describe the content of a webpage (not a programming language).

HTML consists of elements that describe different type of content: paragraph, links,heading, images, videos etc web browsers understand HTML and render HTMl code as websites.

In HTMl, each element is made up of 3 parts:

  1. The opening tag
  2. the closing tag
  3. The actual element

you can learn more at the MDN web docs
